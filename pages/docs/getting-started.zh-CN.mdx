# 如何更好的使用 React

本教程旨在帮助你更好的使用 `React`，它不是一个 `React` 入门教程，所以你需要对 `React` 有一定的了解。

我们将尝试 `React Hooks` 的一些高级用法，所以你需要对 `React Hooks` 有一定的了解。

通过本教程，你将学到如何更好的使用 `React Hooks`，比如：

- 如何更好的封装你的业务逻辑。
- 如何避免不必要的组件渲染。
- 如何在一些业务场景使用 `React Hooks` 优化你的应用体验。

因此，你将会更好的控制你的组件状态，避免不必要的组件渲染，在合适的地方选择合适的 `React Hooks`。

一次性解决那些困扰你很久的 `React Hooks` 问题。
