# 状态是如何影响组件渲染的？

前面的章节我们提到了 `UI = F(state)` 这个公式，知道了组件的渲染是由状态决定的，那么状态具体是如何影响组件渲染的呢？

`React` 组件可以接收外部传入的 `props`，也可以有自己的 `state`，`props` 和 `state` 都属于组件的状态，当状态发生变化时，组件会重新渲染。
